# 🚗 Smart Check Engine

A fullstack web app that helps everyday drivers to understand their OBDII check engine codes when their engine light lit up,explained in plain English with severity ratings and estimated CAD repair costs.

![Smart Check Engine Demo](demo.gif)

## ✨ Features
- 🔍 Search any OBD-II code instantly
- 🟢 Green / 🟡 Yellow severity badges (Driveable vs Check Soon)
- 💵 Estimated repair cost ranges in CAD
- 🔧 Common parts involved for each code
- 📋 Browse 227+ common codes

## 🛠 Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React, React Router |
| Backend | Python, Flask |
| Database | SQLite |
| Styling | Custom CSS |

## 🚀 How to Run Locally

**Backend:**
```bash
cd backend
source venv/bin/activate
python3 app.py
```
**Frontend:**
```bash
cd frontend
npm install
npm start
```
